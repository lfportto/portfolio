# Professional Portfolio Website

📄 [Versão em português](README.md)

## Project Description

This project consists of the development of a responsive professional portfolio for showcasing professional experience, technical skills, projects, and contact information. The application was developed as a Single Page Application (SPA) using mainly React and TailwindCSS, with a focus on smooth navigation, user experience, and responsiveness across different devices. The goal is to centralize and organize professional information, projects, and relevant links within a single platform.
The website can be accessed by clicking [here](https://portfolio-luis-felipe-porto.vercel.app/#contact).

## 🎯 Objective

The project was developed with the following objectives:

* Showcase professional experience and career journey;
* Centralize projects developed in the fields of Data, Business Intelligence, and Automation;
* Provide access to resume and contact channels;
* Create a modern and responsive interface for desktop and mobile viewing;
* Serve as an online portfolio for professional sharing and networking.

## Main Features

### 🔹 Single Page Navigation (SPA)

* Section-based navigation using smooth scrolling;
* Responsive menu for desktop and mobile devices;
* Visual navigation indicators.

### 🔹 Hero Section

* Professional introduction;
* Links to projects, contact information, and resume;
* Animated visual elements.

### 🔹 About

* Professional summary;
* Highlight of technologies used;
* Additional information about experience and professional profile.

### 🔹 Projects

* Showcase of featured projects;
* Interactive cards with summarized information;
* Modals with detailed descriptions of each project;
* Links to repositories and published dashboards.

### 🔹 Professional Experience

* Interactive timeline with professional history;
* Information about positions, companies, and responsibilities;
* Highlight of technologies used in each role.

### 🔹 Technical Skills

* Skills organized by category;
* Visual display of key tools and technologies.

### 🔹 Contact

* Links to LinkedIn, GitHub, and email;
* Quick access to professional channels.

### 🔹 Light and Dark Theme

* Toggle between light and dark modes;
* User preference persistence across sessions.

### 🔹 Responsiveness

* Layout adapted for desktop, tablets, and smartphones;
* Typography, spacing, and component adjustments for different screen sizes.

## Deployment

The application is hosted on Vercel and integrated with GitHub for continuous deployment.

Update workflow:

1. Changes are pushed to the GitHub repository;
2. An automatic build is triggered by Vercel;
3. The new version is published to production.
